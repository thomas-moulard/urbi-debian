\chapter{Segway RMP}
\label{sec:segway-rmp}

The Segway \dfn{RMP} (Robotic Mobility Platform) is a robotic platform based
on the Segway Personal Transporter.  \urbi for Segway RMP can be downloaded
from \url{http://www.urbiforge.com/index.php/Robots/Segway}.

\section{RMP}
\defObject{RMP}

The UObject RMP provides the interface from \urbi.

\subsection{Instantiation}

\begin{urbiunchecked}
var rmp = RMP.new(\var{period});
\end{urbiunchecked}

\var{period} is the period, in seconds, at which orders will be sent and
coders will be read.

\subsection{Slots}

\paragraph{Raw effector variables}

\begin{urbiscriptapi}
\item[forwardSpeed] forward speed in meters per second.  Can be read to
  (behaves as a sensor), or written to (effector).
\item[yawSpeed] yaw speed in radian per second.  Can be read and written
  to.
\end{urbiscriptapi}

\paragraph{Raw sensors}
Current speed and coder values.
\begin{urbiscriptapi}
\item[leftSpeed]
\item[rightSpeed]
\item[forward]
\item[yaw]
\item[left]
\item[right]
\end{urbiscriptapi}

\paragraph{Calibration}

\urbi unit per coder unit coefficients.

\begin{urbiscriptapi}
\item[forwardSpeedCoeff]
\item[yawSpeedCoeff]
\item[leftSpeedCoeff]
\item[rightSpeedCoeff]
\item[forwardCoeff]
\item[yawCoeff]
\item[leftCoeff]
\item[rightCoeff]
\end{urbiscriptapi}

The file \file{usegway.u} contains approximate values for meter, seconds,
radian units.


\section{GSRAPI compliance}

The file \file{rmp.u} will initialize the RMP UObject and give it a Gostai
Standard Robotic API compliant interface.
